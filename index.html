HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página web. É o bloco de construção mais básico da web e é usado para criar páginas web e aplicações online.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos escritos em HTML ou XML. Ela define como os elementos de uma página web devem ser exibidos, permitindo o controle do layout, cores, fontes, espaçamento e outros aspectos visuais.
